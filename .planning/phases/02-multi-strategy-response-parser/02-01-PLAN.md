<feature>
  <name>Python AST Function Call Extraction</name>
  <files>src/perplexity_web_mcp/api/strategies/python_ast.py, tests/test_python_ast_strategy.py</files>
  <behavior>
    Extract function calls from Python code blocks using AST parsing

    Cases:
    - "search('query')" -> [{'name': 'search', 'args': ['query'], 'kwargs': {}}]
    - "read_file('test.txt', encoding='utf-8')" -> [{'name': 'read_file', 'args': ['test.txt'], 'kwargs': {'encoding': 'utf-8'}}]
    - "calculate(1, 2, 3)" -> [{'name': 'calculate', 'args': [1, 2, 3], 'kwargs': {}}]
    - "process(data={'key': 'value'})" -> [{'name': 'process', 'args': [], 'kwargs': {'data': {'key': 'value'}}}]
    - "invalid python {}" -> []
    - "print('hello')\nsearch('world')" -> [{'name': 'print', ...}, {'name': 'search', ...}]
  </behavior>
  <implementation>
    1. Use ast.parse() to parse code string
    2. Walk AST tree with ast.walk()
    3. Extract ast.Call nodes
    4. Extract function name from node.func.id
    5. Parse arguments with ast.literal_eval
    6. Return list of call dictionaries
    7. Handle SyntaxError by returning empty list
  </implementation>
</feature>
