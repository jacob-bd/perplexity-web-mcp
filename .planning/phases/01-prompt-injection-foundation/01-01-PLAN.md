  <action>
  Create a new module for converting Anthropic tool schemas to Python function definitions.

  Implement two main functions:
  1. format_tool_as_python(tool: dict) -> str
     - Extract name, description, and input_schema from tool dict
     - Build function signature with typed parameters
     - Generate Google-style docstring with Args section
     - Handle required vs optional parameters (optional get = None default)
     - Map JSON types to Python types (string -> str, integer -> int, boolean -> bool, array -> list, object -> dict)

  2. format_tools_list(tools: list[dict]) -> str
     - Take list of tools and format each one
     - Join with blank lines between functions
     - Add header comment if needed

  Use Python standard library only (no external dependencies).
  Reference the pattern from RESEARCH.md lines 60-93.

  Avoid: Complex type annotations that might confuse models. Keep signatures simple.
  </action>
