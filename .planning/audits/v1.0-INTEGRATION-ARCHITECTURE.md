## Data Flow Summary

### Request → Response (No Tool Results)
1. User sends tools + query
2. **Phase 3:** Validate (no results to check)
3. **Phase 4:** Extract results (empty)
4. **Phase 1:** Inject tools into prompt
5. Send to Perplexity
6. **Phase 2:** Parse response
7. **Phase 3:** Transform to tool_use blocks
8. **Phase 5:** Record metrics
9. Return tool_use response

### Tool Result → Continuation
1. User sends tool_result message
2. **Phase 3:** Validate result matches prior tool_use
3. **Phase 4:** Extract tool results from history
4. **Phase 1:** Inject tools + results into prompt
5. Send to Perplexity (with context)
6. **Phase 2:** Parse response (likely no tools this time)
7. **Phase 3:** Transform (text-only if no tools)
8. **Phase 5:** Record metrics
9. Return final response

### Graceful Degradation Path
1. Any step → Exception or low confidence
2. **Phase 5:** Log warning, record failure metrics
3. **Phase 2:** Return confidence: 0.0
4. **Phase 3:** Filter out low confidence → text-only
5. User receives conversational response (no crash)
