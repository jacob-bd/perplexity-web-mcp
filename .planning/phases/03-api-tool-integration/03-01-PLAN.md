1. Add helper function before create_message endpoint (around line 700):
   ```python
   def transform_to_tool_use_blocks(
       parse_result: dict,
       answer: str,
       confidence_threshold: float = 0.7
   ) -> tuple[list[dict], str]:
       """Transform parser output to tool_use content blocks.

       Args:
           parse_result: Output from parse_response()
           answer: Original model answer text
           confidence_threshold: Minimum confidence to return tool_use

       Returns:
           (content_blocks, stop_reason) tuple
       """
   ```

2. Logic:
   - If parse_result["tool_calls"] is empty -> return ([{"type": "text", "text": answer}], "end_turn")
   - If parse_result["confidence"] < confidence_threshold -> same as above, log: "Tool calls detected but confidence {conf} < threshold {thresh}"
   - If confidence >= threshold:
     - content_blocks = []
     - If answer.strip() is not empty, add text block first: {"type": "text", "text": answer}
     - For each tool_call in parse_result["tool_calls"]:
       - Generate tool_use_id: f"toolu_{uuid.uuid4().hex[:24]}"
       - Append ToolUseBlock(id=tool_use_id, name=tool_call["name"], input=tool_call["arguments"]).model_dump()
     - Return (content_blocks, "tool_use")

3. Replace lines 845-860 (current parse_response call and TODO) with:
   ```python
   if body.tools:
       try:
           parse_result = parse_response(answer)
           content_blocks, stop_reason = transform_to_tool_use_blocks(parse_result, answer)

           if stop_reason == "tool_use":
               logging.info(
                   f"Returning {len([b for b in content_blocks if b.get('type') == 'tool_use'])} tool_use blocks "
                   f"using {parse_result['strategy']} strategy (confidence: {parse_result['confidence']:.1f})"
               )
           else:
               logging.info(f"No tool calls returned (confidence: {parse_result.get('confidence', 0):.1f}, threshold: 0.7)")
       except Exception as parse_error:
           logging.warning(f"Response parsing failed: {parse_error}")
           content_blocks = [{"type": "text", "text": answer}]
           stop_reason = "end_turn"
   else:
       content_blocks = [{"type": "text", "text": answer}]
       stop_reason = "end_turn"
   ```

4. Update response construction (line ~866-873):
   - Change from: "content": [{"type": "text", "text": full_response}]
   - To: "content": content_blocks (after adding citations to last text block if present)
   - Change from: "stop_reason": "end_turn"
   - To: "stop_reason": stop_reason

5. Handle citations: If content_blocks has text blocks, append citations to the LAST text block's text, not to a tool_use block.

Use uuid.uuid4().hex[:24] for tool IDs (already imported at top of file). Do NOT use jsonwebtoken or other libraries.
